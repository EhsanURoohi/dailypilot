# DailyPilot
DailyPilot is a cloud-native software startup building lightweight automation tools for students and small teams. 

## Project Overview
DailyPilot Tasks is a secure, minimal web application designed to help users track their daily productivity without the distractions of bloated task managers.

## Team Members
- **Ehsan** (Project Lead/Repo Owner)
- **Melkias Gobai** (GCP Owner/Frontend)

## Core Features
- User Authentication: Secure login for isolated task management.
- Task Management (CRUD): Create, Read, Update, and Delete daily tasks.
- Progress Dashboard: Visual overview of productivity.
- Secure Infrastructure: Integration with Google Secret Manager and automated CI/CD.

## Architecture & DevSecOps
This project utilizes modern DevSecOps practices, including:
- Containerized deployment to Google Cloud Run.
- Automated CI/CD pipelines via GitHub Actions.
- Security-first IAM configuration and secret management.
